L.E.S - Project Components

Components Used in the Lab Embedded System Project

Microcontroller & Core

  • Arduino UNO - Main microcontroller for controlling the entire system
  • Custom PCB Board - Designed to hold and organize all components

Communication Modules

  • HC-05/HC-06 - Bluetooth communication
  • ESP-01/ESP8266 - Wi-Fi/IoT connection
  • SIM800L - GSM module for SMS/call

Supporting Tools

  • Breadboard - For prototyping
  • Jumper Wires - For connections
  • Power Supply - Stable 5V/3.3V power

Sensors

Component Function
DHT11 / DHT22 Temperature & Humidity sensor
BMP280 / BME280 Pressure, temperature, and humidity
MQ3 Gas and alcohol detection
HC-SR04 Ultrasonic distance measurement
LDR Light intensity sensor
PIR Sensor Motion detection

Output Modules

Displays

  • 16x2 LCD (I2C) - Display text
  • OLED Display - Display graphics and data

Actuators

  • LEDs - Indicators and lighting
  • Buzzer - Sound alert
  • Relay Module - Control high-voltage devices

Motors

  • DC Motors + Motor Driver - For robotic movement
  • Servo Motors - Precise angular movement

Project Implementation Steps

  1. Idea Selection

    Choose a smart, functional idea suitable for embedded systems (e.g. sensor-based automation or control systems)

  2. System Design

    Draw a block diagram showing inputs (sensors), processing (Arduino), and outputs (displays, motors, buzzers). Plan wiring and communication protocols (I2C, PWM, Serial, etc.)

  3. Prototyping on Breadboard

    Test individual sensors and modules. Use Serial Monitor for debugging. Validate working logic before final build.

  4. Arduino Programming

    Write code using Arduino IDE. Use libraries like LiquidCrystal_I2C.h for LCD, Servo.h for servo motors, SoftwareSerial.h for GSM/Bluetooth.

  5. Custom PCB Design

    Use tools like Proteus or EasyEDA. Route power lines and signal paths efficiently. Add silkscreen labels for ports and components.

  6. Soldering and Assembly

    Solder components onto the PCB. Verify connections and upload the code. Test the system with full integration.

  7. Final Enclosure

    Design a box or acrylic casing. Ensure proper openings for USB, buttons, LCD, power, etc. Secure internal wiring and manage heat if needed.
